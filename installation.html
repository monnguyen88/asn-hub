Local Server Installation

There are the three things to setup & run the theme.

  • 1. Install
  • 2. How to Setup Theme Code Folder
  • 3. Run the theme into browser

Step-1 : Install

Unzip the downloaded zip file themeforest-xxxxx.zip folder.

Step-2 : How to Setup Theme Code Folder

Open folder named main-file folder.

Here you will find assets folders of all files.

Make sure you upload the required files/folders listed below:

  • css - Extra Stylesheets Folder plus Main Stylesheet File
  • image - Images Folder
  • Js - Javacripts Folder

Step-3 : Run the theme into browser

You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.
